# Comandos Básicos Git



# Windows           

- dir - diretórios
- cd -  diretório específico + local
- cd .. - voltar para o diretório
- cls  ou ctrl + l - limpar
- tab + letra - atalho
- mkdir - criar diretório
- echo - criar arquivo
- del - deletar arquivo
- rmdir - deletar repositório



### Linux

- ir - diretórios
- cd - diretório específico + local
- cd .. - voltar para o diretório
- clear - limpar
- tab + letra - atalho
- echo - criar arquivo 
- del - deletar arquivo
- rm -rf - deletar repositório
